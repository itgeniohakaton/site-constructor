Как пользоватся этим приложением?

Главные предметы в параграфе этого конструктара:

  • Img link
  • Paragraph text
  • background
  • font family
  • img alignment
  • img size
  • ADD
  • Delete all
  • Delete component
  • Delete last component
  • Download

Подробный разбор:

  • Img link нужен для добавление ссылки на картинку который хочешь добавить.
  • Paragraph text нужен для добавление текста
  • background нужен для цвета фона "ОБЕЗАТЕЛЬНО".
  • font family нужен для выбора формы текста "ОБЕЗАТЕЛЬНО".
  • img alignment нужен для выбора сторону картинки "ОБЕЗАТЕЛЬНО".
  • img size нужен для выбора размера картинки "ОБЕЗАТЕЛЬНО".
  • ADD нужен для показа того что появится в меню "ОБЕЗАТЕЛЬНО".
  • Delete all может удалить всё что вы добавили.
  • Delete component может удалить компонент который вы хотите.
  • Delete last component может удалить последний елемент который вы добавили.
  • Download-скачать.
